Программа "Самый умный"
------------------------
М. Гладких. 2010 г.

При запуске программы будет показано окно настроек.

== Вкладка "Управление" ==
С помощью этой вкладки можно настроить клавиши, с помощью
которых будут производиться операции с полем.

- распределение цветов по полю (по умолчанию - клавиша F2)
- показ всего поля (по умолчанию - клавиша F5)
- скрытие неоткрытых ячеек (по умолчанию - клавиша F9)
- развернуть окно программы на весь экран (по умолчанию - клавиша F11)
- вызвать окно настроек

== Вкладка "Цвета" ==
На этой вкладке настраиваются цвета команд, нейтральный цвет и цвет
неоткрытых ячеек

== Вкладка "О программе"
На этой вкладке находится информация об авторе сего произведения.
